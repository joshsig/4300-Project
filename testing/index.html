COMP4300 Project by Darwin, Andrii, and Josh

This project has two sections.

  1. The Game

    This game shows users levels of increasing complexity, giving them a visualization of common networking architectures. Users are enabled to alter the networks and learn as they play!

  2. The Simulation

    The simulation takes the user's network and finds DNS servers they are connected to. This is then displayed to the user in a visual format.

The game and simulation are designed to be educational and fun. We hope you enjoy them!
